GitHub Pages로 블로그 시작하기

2025년 11월 13일

GitHub Pages는 깃허브에서 제공하는 무료 호스팅 서비스입니다. 정적 웹사이트, 블로그, 포트폴리오 등을 손쉽게 배포할 수 있죠.

별도의 서버나 복잡한 설정 없이도 몇 분 만에 웹사이트를 만들 수 있습니다. 게다가 깃허브의 빠르고 안정적인 글로벌 CDN의 혜택을 받을 수 있어요!

왜 GitHub Pages인가?

2025년 11월 13일

무료 호스팅: 깃허브 계정만 있으면 누구나 무료로 사용할 수 있습니다.

간편한 배포: git push만 하면 자동으로 배포됩니다.

커스텀 도메인: 별도로 구매한 도메인도 연결할 수 있습니다.

HTTPS 지원: 무료 SSL 인증서가 자동으로 제공됩니다.
